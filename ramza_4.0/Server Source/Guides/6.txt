Algunos de estos comandos solo pueden ser usados por administradores:
/hechar (hecha a un jugador)
/online (muestra quien esta online)
/ban (banea un jugador)
/banip (banea una ip)
/warpto # (transporta a un mapa)
/warpmeto ..... (transporta hacia un jugador)
/warptome ..... (transporta a un jugador)
/reiniciar (reinicia el mapa)
/editarmapa (abre el editor de mapas)
/editartienda (abre el editor de tiendas)
/editaritem (abre el editor de items)
/editarnpc (abre el editor de npcs)
/inv (abre el inventario)
/info (ver las estadisticas)
/estado (ver tus estadisticas)
/fps (muestra los cuadros por segundo)
/entrenar (entrenar habilidades)
/party (formar una party)
/unirse (unirse a una party)
/dejar (dejar una party)
/actualizar (actualiza la pantalla del jugador)
/editarhechizo (abre el editor de hechzos)
/sprite # (cambia el sprite)
//spritedejugador # (cambia el sprite de otro jugador)
/loc (muestra tu locacion)
/admin (muestra los comandos de admin)
/ayuda (muestra los comandos normales)
/editaremoticon (abre el editor de emoticones)
/motd (modifica el mensaje de bienvenida)
/mapas (muestra la lista de los mapas)
/comerciar (comerciar con otro jugador)
/llamaradmin (llamar a un administrador)
/dianoche (cambia entre dia y noche)
